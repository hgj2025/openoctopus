核心框架能力

  Agent 引擎

  - 多模型支持：Anthropic、OpenAI、Gemini、Moonshot、MiniMax、Qwen、VolcEngine、Byteplus、vLLM、Bedrock 等 30+ provider
  - 多 workspace/账号隔离，每个 agent 可有独立 auth 和配置
  - 思考级别控制（basic/medium/high extended reasoning）
  - Bash 工具集成（带审批机制和安全沙箱）
  - Session 持久化和 transcript 管理

  Memory 系统

  - LanceDB 向量数据库长期记忆（extensions/memory-lancedb）
  - 文件型记忆搜索（extensions/memory-core）
  - 自动 recall + 自动 capture（生命周期钩子触发）
  - memory search / memory get CLI 工具

  ---
  基础设施

  - Plugin SDK — registerTool/Channel/Cli/HttpHandler，可扩展
  - Daemon 管理 — launchd (macOS) / systemd (Linux) 后台服务
  - Docker 沙箱 — docker/ 下的 sandbox 镜像
  - Gateway — WebSocket 双向通信，工具分发，auth 限流
  - 多频道架构 — 除 Feishu 外还有 WhatsApp、Telegram、Slack、Discord 等频道插件（代码保留但非主用）

  ---
  主要执行流

  Feishu Webhook → Channel Plugin → Gateway → Agent Engine
    → Model API → Tool 分发（Feishu/Bash/Memory）→ 响应回写 Feishu

  当前主要用途：Feishu 聊天/文档 → AI agent → 读写代码、文档、执行 git 操作。
